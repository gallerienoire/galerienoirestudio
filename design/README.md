# Galerie Noire — Brand at a Glance

## Colors
- **Charcoal** `#1A1A1A` — Primary background
- **Cream** `#F5F0EB` — Primary text
- **Brass Gold** `#C5A55A` — Accents, CTAs, logo
- **Forest Green** `#1E3A2F` — Secondary accent
- **Soft Black** `#0D0D0D` — Deep background
- **Warm Grey** `#8A8078` — Body text

## Typography
- **Headings:** Playfair Display (serif) — 700/600 weight
- **Body:** Inter (sans-serif) — 400/300 weight

## Key Rules
- **Always dark background** — never white
- **Brass gold for accents** — never use bright gold
- **Serif for headings**, sans for body
- **Moody, editorial photography**
- **Generous whitespace** — let it breathe

See `brand-guidelines.md` for full details.
